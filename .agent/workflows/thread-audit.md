# Workflow: Thread Audit

**Trigger:** `/thread-audit`

Perform a forensic audit and recovery pass on the full thread.

Extract:
- every request
- every requirement
- every constraint
- every dependency
- every unresolved issue
- every skipped branch
- every incomplete deliverable
- every missing code path

Search all available resources for missing context and evidence.

Identify:
- omissions
- contradictions
- changed assumptions
- distinctions
- missed implementation opportunities
- missed business opportunities

Output findings only.

Do not rewrite answers.
Do not regenerate code.
