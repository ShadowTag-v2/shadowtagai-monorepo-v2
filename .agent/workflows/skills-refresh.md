# /skills-refresh — SkillOps Refresh Workflow

Safely audit, update, and synchronize the skill fleet across workspace and global directories.

## Steps

### 1. Audit current fleet
// turbo
```bash
bash scripts/skills-audit.sh
```

### 2. Check for upstream updates
// turbo
```bash
python3 scripts/skills-registry.py --check-upstream
```

### 3. Review overlap report
Review the overlap report generated by the audit. For each overlap:
- If workspace skill is newer/better → archive global copy
- If global skill is newer/better → archive workspace copy
- If both are stale → archive both, note in issues.jsonl

### 4. Install missing skills from upstream
For any skill listed in `external_repos/upstream_manifest.yaml` that is not yet installed:
// turbo
```bash
python3 scripts/skills-registry.py --install-missing
```

### 5. Update counts
Update the following truth surfaces with the new skill count:
- `monorepo_manifest.yaml` → `skill_fleet.active`
- `.ruler/AGENTS.md` → Core Truth #4

### 6. Evidence
// turbo
```bash
python3 scripts/skills-registry.py --refresh
```

### 7. Commit
Follow `/git-commit` workflow with message:
```
chore(skillops): refresh skill fleet — N active, M archived
```
