# MCP: Brave Search Server (Native Skill)

## Goal
Query external documentation, library configurations, and competitor intelligence.

## Rules of Engagement (COR.30 Compliance)
1. **Information Verification:** Any code copied from the public internet via search MUST be mathematically traced and validated by `mcp-sequential-thinking` before insertion into the IDE.
2. **No Data Exfiltration:** Under no circumstances may sensitive internal code errors or IP be used as search queries. Strip local contextual identifiers from searches.
