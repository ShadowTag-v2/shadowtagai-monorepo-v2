Phase 1: Corporate Structure (California)
1. Entity Selection: C-Corp vs. LLC
For a high-growth tech company ("ShadowAgai") intending to raise venture capital or issue equity to employees:
Delaware C-Corporation is the industry standard (investors prefer it).
California Corporation is viable if you are 100% CA-based and bootstrap-focused, but you will still pay CA franchise taxes.
Recommendation: If aiming for VC, form a Delaware C-Corp and register it as a foreign entity in California. If bootstrapping forever, a CA LLC or CA Corp accounts for lower initial complexity.
2. Incorporation Steps (California Domestic Corp)
If you choose to incorporate directly in California Pro Per (representing yourself):
Name Availability Search:
Visit CA Secretary of State Business Search.
Ensure "ShadowAgai" is available.
File Articles of Incorporation (Form ARTS-GS):
File Online: Use the bizfileOnline portal.
Cost: ~$100 filing fee (subject to change).
Shares: Authorize a specific number of shares (e.g., 10,000,000 Common Stock).
Statement of Information (Form SI-550):
File within 90 days of incorporation.
List officers (CEO, Secretary, CFO) and directors.
Corporate Bylaws:
Draft internal rules for operation (not filed with the state).
Templates are available online (e.g., Y Combinator's Clerky or Stripe Atlas documents are good references).
EIN (Employer Identification Number):
Apply instantly via the IRS Website.
3. "Doing Business" Compliance
Franchise Tax: Minimum $800/year (First year exemption may apply for Corp, but verify current statutes).
Local Business License: Register with your specific city/county (e.g., San Francisco, Los Angeles).

Phase 2: Intellectual Property (IP) Protection
1. Trademark ("ShadowAgai", "ShadowTag")
Protect your brand identity.
Search: Search the USPTO TESS database to ensure "ShadowAgai" isn't conflicting.
Filing (Pro Per):
File via USPTO TEAS Plus (Electronic).
Cost: ~$250 per class of goods/services.
Class: Likely Class 9 (Software) or Class 42 (SaaS).
Mark Type: Standard Character Mark (protects the name itself, not just the logo).
2. Copyright (Code & Content)
Protect your shadowtag-v2 codebase.
Automatic: Copyright exists the moment you write code.
Registration: Required before you can sue for infringement.
Process:
Visit Copyright.gov.
File Form TX (Literary Work) for source code.
Deposit: Submit the first and last 25 pages of source code (redacted for trade secrets).
Cost: ~$65.
3. IP Assignment (Critical)
Proprietary Information and Inventions Assignment Agreement (PIIAA):
You (as the founder) must assign all code you wrote before incorporation to the company.
Without this, you personally own the IP, not "ShadowAgai," which kills due diligence.
4. Open Source Strategy
Since you are on GitHub (ShadowTag-v2), clarify your license.
Proprietary: usage of LICENSE file saying "All Rights Reserved" or "Proprietary - ShadowAgai".
Open Core: AGPLv3 for the core, commercial license for enterprise features.
