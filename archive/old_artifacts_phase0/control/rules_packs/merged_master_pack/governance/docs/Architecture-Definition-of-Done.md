# Architecture Definition of Done

- Atomic Design is used only for the design-system layer.
- Application code is organized by feature and concern.
- Components have clear boundaries and no hidden logic swamp.
- Stateful and business logic is extracted into hooks or services.
- Route handlers are thin.
- Policies, validation, and permissions are explicit.
- No pointless micro-file fragmentation.
- No giant mixed-responsibility files.
- The repo has one clear way to do things.
