2) Core control-flow (Run1 → Run2 → Run3)

Goal: every generation is produced, then reverse-engineered by Cursor Plan Mode, then challenged by a sibling chain before final return. This bakes explanation + auditability into default operation.

Actors
	•	Chain A (primary)
	•	Chain B (adversarial sibling)
	•	Cursor Plan Mode (auto reverse-engineering step)
	•	Safety Router (Google + Hive)
	•	ShadowTag (writes evidence)

Flow
	1.	Run1 (Chain A, “original”)
	•	LangChain agent builds plan → calls Nowgrep → fuses results → drafts code/answer.
	2.	Run2 (Plan Mode on A’s output, “explain”)
	•	Cursor Plan Mode runs on the produced artifact → generates a step-by-step explanation & implicit test plan.
	•	This Run2 explanation is what A publishes to B.
	3.	Run3 (Chain B, “counter-run”)
	•	Chain B treats Run2 as spec, re-executes a fresh Run1 on the same query using B’s tools (can vary retriever/reranker).
	•	B produces its own artifact + deltas vs A.
	4.	Adjudication
	•	Orchestrator compares A’s Run1 artifact vs B’s Run3 artifact using a reranker + rule checks (and quick lint/tests from the Run2 test plan).
	•	Winner (or merged) result is moderated (Google/Hive) → ShadowTag attested → returned to UI/CLI.

Why this works (solo-founder practical):
	•	Forces explanations by default (Run2) without you writing bespoke docs.
	•	Introduces competitiveness (Run3) to reduce single-chain blind spots.
	•	Keeps latency reasonable by reusing memory (GPTRAM) and limiting heavy steps to 3 total passes.

3.2 Run1/Run2/Run3 (very small glue)

# services/orchestrator-langchain/run_cycles.py
from .chains import chain_a, chain_b, cursor_plan_mode
from ..moderation_safety.router import moderate
from ..shadowtag_audit.writer import attest

async def run_all(query: str, user_id: str | None, mode="default"):
    # Run1 – Chain A original
    a_out = await chain_a.run1(query, mode=mode)

    # Run2 – reverse-engineer using Cursor Plan Mode
    a_explain = await cursor_plan_mode.reverse_engineer(a_out.artifact)

    # Run3 – Chain B recreates from A's explanation
    b_out = await chain_b.run3_from_spec(query, spec=a_explain.plan, mode=mode)

    # Adjudicate (simple: prefer passing tests + higher reranker score)
    winner = await _adjudicate(a_out, b_out, a_explain.tests)

    # Safety pass (Google+Hive)
    safe = await moderate(winner.content, media=winner.media)

    # Attest evidence (hashes, model, versions, tests, moderation)
    receipt = await attest(user_id=user_id, query=query, result=winner, moderation=safe)

    return {
        "answer": safe.content,
        "explanation": a_explain.plan,
        "diff": winner.diff,        # deltas between A and B if any
        "audit_receipt": receipt    # ShadowTag reference
    }

async def _adjudicate(a_out, b_out, tests):
    # 1) quick tests from Run2
    a_ok = await tests.run(a_out.artifact)
    b_ok = await tests.run(b_out.artifact)
    # 2) reranker score (higher is better)
    a_score = await b_out.reranker.score(a_out.support)
    b_score = await a_out.reranker.score(b_out.support)
    # policy: prefer passing tests; tie → higher score; else merge
    if a_ok and (a_score >= b_score or not b_ok): return a_out
    if b_ok and (b_score > a_score or not a_ok): return b_out
    return await _merge(a_out, b_out)

async def moderate(content: str | None, media: list[str] | None = None):
    g = await google_text_scan(content) if content else {"ok": True}
    h_txt = await hive_text_scan(content) if content else {"ok": True}
    h_img = await hive_media_scan(media) if media else {"ok": True}

    verdict = all(x.get("ok", False) for x in (g, h_txt, h_img))
    if not verdict:
        # mask or rephrase; attach labels
        content = _redact(content, [g, h_txt, h_img])
    return {"ok": verdict, "content": content, "labels": [g, h_txt, h_img]}
