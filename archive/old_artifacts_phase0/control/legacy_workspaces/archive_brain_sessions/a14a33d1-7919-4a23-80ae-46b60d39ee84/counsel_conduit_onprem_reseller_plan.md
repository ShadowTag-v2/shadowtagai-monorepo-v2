## 2. The Tech Stack: The Localized Proxy Pattern ("Deploy-in-VPC")
By reversing the deployment model, we achieve the holy grail of enterprise software: zero data liability paired with absolute lock-in.

### 2.1 The "Blind Telemetry" Architecture
1.  **The Master Key Engine (Delaware C-Corp):** CounselConduit holds the massive Enterprise API contracts with Anthropic and Google. We are the volume buyer.
2.  **The Localized Edge Node (Law Firm Network):** We provide the law firm with a contained software package that they spin up within their own private cloud (AWS VPC, Azure, or on-premise servers).
    *   **Tech Stack:** A lightweight, containerized Go or Node.js proxy delivered via **Docker**, paired with a **Next.js** frontend UI that seamlessly mimics the native Claude/Gemini chat experience.
3.  **The Routing Loop:** When a client accesses the firm's private "Magic Link," the device connects directly to the firm's localized edge node. The prompt is fired from the firm's server to Anthropic/Google using our master keys.
4.  **The Local CRM Hook:** The response hits the client and is simultaneously POSTed straight into the firm's Clio/PracticePanther CRM using the firm's own local API keys.
5.  **The Telemetry Feed:** The *only* data that pings back to CounselConduit headquarters is purely mathematical billing telemetry: *"Firm #4102 just burned 8,000 Claude 3.5 Sonnet input tokens."*

### 2.2 Why This is the Holy Grail for Investors
*   **Absolute Data Immunity:** We do not host endpoints or possess databases of client transcripts. The Delaware C-Corp is utterly immune to subpoenas for client data.
*   **Instant AmLaw 200 Penetration:** Big Law defense firms undergo brutal IT security audits. By deploying on their own infrastructure, CounselConduit skips the standard 9-month vendor security review. "The data never leaves your building."
*   **Pristine Cap Table:** A highly investable, standard Delaware C-Corp that Tier-1 Venture Capital (a16z, Founders Fund) can wire $50M into without touching a drop of fiduciary liability.
