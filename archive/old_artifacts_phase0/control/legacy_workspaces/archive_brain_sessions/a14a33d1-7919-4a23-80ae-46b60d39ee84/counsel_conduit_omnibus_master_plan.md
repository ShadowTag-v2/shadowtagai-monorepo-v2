## 2. The Core Mechanic: Asymmetric AI Generation ("The Bifurcated Payload")
We utilize **Structured JSON Outputs** to split the reality of a single prompt from a client into two distinct, asymmetric paths:

1.  **`client_facing_paralegal`:** The client receives an empathetic, strictly non-advisory, factual intake acknowledgment on their phone/browser. *This interface is ephemeral and vanishes after 60 minutes, ensuring zero UPL data traces.*
2.  **`attorney_facing_strategy`:** The heavy, uncensored "Claude-Law" analysis (citing case law, statutes, and tactical leverage) bypasses the client entirely and is injected straight into the lawyer's CRM matter file.

### The Revenue Engine: Triple-Dip Billing
By creating this distinct separation, the attorney creates three distinct billable events from a single client action:
1.  **The Intake:** Billing the client for the time spent interacting with the portal.
2.  **The Review:** Billing attorney time reviewing the generated "Claude-Law" analysis.
3.  **The Execution:** Billing attorney time drafting pleadings directly extracted from the analysis.

## 3. The Tech Stack: The On-Premise Reseller Vault
To achieve absolute data immunity, we refuse to host the proxy endpoints or the data itself.

1.  **The Master Key Engine (Delaware C-Corp):** CounselConduit holds the massive Enterprise API contracts with Anthropic and Google. We are the wholesale buyer.
2.  **The Localized Edge Node (The "Deploy-in-VPC" Law Firm Network):** We provide the law firm with a contained software package (a lightweight Go/Node.js proxy natively wrapped in a **Docker** container) paired with a **Next.js** frontend UI. They spin this up entirely within their own private cloud (AWS VPC, Azure, or on-prem servers).
3.  **The "Blind Telemetry" Loop:** The client hits the firm's private "Magic Link" domain. The firm's server fires the prompt to Anthropic/Google using our master keys. The response is parsed; the paralegal fluff goes to the client, the strategy goes directly to **Clio / PracticePanther / MyCase** using the firm's local CRM keys.
4.  **Zero Transcripts, Only Mathematics:** The *only* data that pings back to CounselConduit headquarters is purely mathematical billing telemetry (e.g., *"Firm #4102 just burned 12,000 token inputs"*).

### Why This is the Holy Grail for Investors:
*   **The Pristine Cap Table:** Because we hold zero data, we do not need complex Offshore Data Havens (Panama/Philippines) to avoid US subpoenas. We operate as a highly investable, standard **Delaware C-Corp**.
*   **Instant AmLaw 200 Penetration:** Big Law defense firms undergo brutal SOC-2 audits. By deploying on their own infrastructure, CounselConduit skips the standard 9-month vendor security review. The client data literally never touches a server outside their ownership.
