# Launch Packet Constitution v9

Antigravity / VS Code must start from one packet.

## Packet contents
- authority snapshot
- top authority atoms
- active tasks
- recent summaries
- drift reports
- monorepo canonical repo-root truth

## Why
This prevents:
- waking up from stale repo assumptions
- using old roots for merged repos
- letting local code/docs redefine current standards or repo topology

## Rule
Authority memory decides what is true.
Monorepo truth decides where live code truth lives.
Codebase decides what must change.
