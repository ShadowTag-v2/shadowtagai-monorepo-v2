# Verdict Systems - Executive Function Replacement Platform

## Mission

Verdict Systems exists to **duplicate and perfect human executive function across domains**—helping individuals, families, schools, workplaces, and professionals move through life without dropping critical tasks. Our core engine ("**Schiznit**") serves as a universal task and life-orchestration layer, ensuring deadlines are met, distractions are managed, and productivity flows naturally.

---

## Core Concept

Verdict Systems combines:

- **Tile-based urgency system** (🟢 green → 🟡 yellow → 🔴 red → ⚫ critical) that escalates with time sensitivity
- **Lockout protocols** to block non-essential apps and force focus until tasks are completed
- **Seamless integration** across phones, desktops, vehicles, and smart homes
- **Built-in assistance layers** (AI tutor, admin oversight, reminders, professional tools)
- **Designed to be an aid, never an impediment**, with user/admin override as needed

## Key Features

### 🎯 Urgency Tile System

Tasks automatically calculate urgency based on:
- Time to deadline
- Priority level
- Task type

**Urgency Levels:**
- 🟢 **GREEN**: >24 hours remaining, low priority
- 🟡 **YELLOW**: 6-24 hours remaining, medium priority
- 🔴 **RED**: 1-6 hours remaining, high priority
- ⚫ **CRITICAL**: Past deadline, lockout active

### 🔒 Lockout Protocols

Enforce focus by blocking distracting apps:

**Lockout Modes:**
- `NONE`: No restrictions
- `SOFT`: Warnings and reminders only
- `MODERATE`: Block non-essential apps
- `STRICT`: Full lockout until completion
- `EMERGENCY_ONLY`: Only emergency functions

**Override Capabilities:**
- Admin/parent/teacher can override
- Emergency situations
- Medical necessity
- Logged for audit trail

### 🤖 AI Assistance Layer

**School Vertical:**
- AI tutor provides hints, not solutions
- Step-by-step guidance
- Concept explanations
- Limited hints before requiring break

**Other Verticals:**
- Smart scheduling suggestions
- Productivity insights
- Motivational messages
- Task breakdown assistance

### 👥 Admin Oversight

**Role-Based Access:**
- **Parents**: Oversee children's tasks, set lockouts, approve completion
- **Teachers**: Assign work, grade submissions, provide feedback
- **Managers**: Track team productivity, enforce focus sessions
- **Caregivers**: Monitor medication, safety alerts, daily routines

## API Quick Start

### Installation

```bash
# Clone repository
git clone https://github.com/ehanc69/ShadowTag-v2-fastapi-services.git
cd ShadowTag-v2-fastapi-services

# Install dependencies
pip install -r requirements.txt

# Run Verdict Systems API
uvicorn src.verdict_systems.api.main:app --reload --port 8001
```

### API Documentation

Once running, access:
- **Swagger UI**: http://localhost:8001/docs
- **ReDoc**: http://localhost:8001/redoc

### Example: Create School Assignment

```bash
curl -X POST http://localhost:8001/school/assignments \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Chapter 5 Homework - Quadratic Equations",
    "description": "Complete problems 1-20, show all work",
    "deadline": "2025-11-18T14:00:00Z",
    "estimated_duration_minutes": 60,
    "priority": 4,
    "user_id": "student_12345",
    "subject": "Algebra II",
    "teacher_id": "teacher_jones",
    "assignment_type": "homework",
    "ai_tutor_enabled": true,
    "enable_lockout": true,
    "blocked_apps": ["instagram", "tiktok", "youtube"]
  }'
```

### Example: Get Dashboard

```bash
curl http://localhost:8001/dashboard/student_12345
```

Returns:
```json
{
  "user_id": "student_12345",
  "urgency_summary": {
    "green": 3,
    "yellow": 2,
    "red": 1,
    "critical": 0
  },
  "next_deadlines": [...],
  "lockout_status": {
    "lockout_active": false,
    "blocked_apps": []
  },
  "completed_today": 4
}
```
