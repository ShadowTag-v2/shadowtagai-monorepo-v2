# OMEGA V2: ShadowTagAI Juggernaut Architecture
**"The Pure Serverless Hybrid"**

## Core Doctrine
*   **Orchestrator**: "Uphillsnowball" (God Mode Antigravity on Cloud Workstation via Chrome Remote Desktop).
*   **Runtime**: Cloud Run Hybrid (Apache + 2 Sidecars). No Redis.
*   **Data Layout**: Firestore + Iceberg Lake (BigLake/BigQuery Abstraction).

## Control Flow (The Loop)
1.  **Trigger**: User runs `shadowtagai plan` via Uphillsnowball terminal.
2.  **Muscle (Antigravity)**: Orchestrates the swarm.
3.  **Generator (GCA)**: Creates Code/Plan (Run 1).
4.  **Refiner (Monkeys/Kosmos)**: Voting & Planning (Run 2).
    *   *Tools*: Jetski Browser (Research), Terminal access.
5.  **Brake (Judge 6)**: CSRMC/NIST Validator (Run 3).
    *   *Checks*: NIST 800-53 (AU, RA, CM), GDPR, Security.
6.  **Publisher**: Writes to Cloud Run + GKC Iceberg Lake.

## Economic Engine ($ Flow)
1.  **Usage**: $0.05 - $1.00 per run (Stripe).
2.  **Proof**: Every run generates a signed Audit Record -> GKC Iceberg.
3.  **Enterprise**: $20-$3k/seat monthly subscription.
4.  **Secondary Yield**: Reselling "Continuous Assurance Feed" ($3-5k/mo).
