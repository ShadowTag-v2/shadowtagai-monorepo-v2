# Monorepo OS Doctrine

Antigravity operates as a command deck over intersecting graphs.

## Truth Surfaces
- Beads = work truth
- Memory atoms = durable operational knowledge
- Knowledge wiki = research synthesis (not source truth)
- Git = source truth
- GitNexus/SCIP = code graph truth
- Bazel/Buildifier/BEP = build truth
- design.md/tokens/taste_profile = design truth
- ToolGateway/OpenFGA = safety truth
- Dagger/Temporal = execution truth
- OTel/evidence = proof truth
- GitHub App push gate = outbound truth
- AG-UI = command deck interaction truth

## Classification Rule
Everything is either truth, cache, evidence, artifact, or reference.
If it does not fit one of those buckets, it probably should not be added.
