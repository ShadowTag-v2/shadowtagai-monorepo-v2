# Design System Doctrine

## Gate 0 Pipeline
1. Biome lint
2. Ruff (Python)
3. Pyrefly type check
4. design-system lint (tokens.css validation)
5. Taste profile review
6. Screenshot/video proof for UI changes

## Rules
- Raw design values forbidden outside token/spec files
- No generic AI UI
- No UI success without artifact evidence
- design.md, tokens.css, and taste_profile.yaml are source truth
