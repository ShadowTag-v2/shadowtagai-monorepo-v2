# Knowledge Compiler Doctrine

The knowledge vault compounds research. It is synthesis, not source truth.

## Pipeline
1. Ingest raw documents → data/knowledge_raw/ (gitignored)
2. Compile into Markdown → knowledge/vault/
3. Update raw_manifest.yaml
4. Run wiki-health.sh
5. Promote durable findings → .memory/atoms/
6. Create Beads for executable work
