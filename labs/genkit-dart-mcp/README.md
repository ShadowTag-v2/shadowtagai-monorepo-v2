# Genkit Dart MCP Server

Lab scaffold for building full-stack AI apps with Dart and Flutter using Genkit.

> Part of the UphillSnowball monorepo labs.

## Reference

- [Genkit Dart announcement](https://x.com/googledevs/status/2036548739566018758)
- Model-agnostic API supporting Google, Anthropic, OpenAI
- Cross-platform: Web, Mobile, Desktop, Server

## Usage

```bash
dart run bin/genkit_dart_mcp.dart
```
