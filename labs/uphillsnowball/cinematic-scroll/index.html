Cinematic Scroll

100% Google-native pipeline. Nano Banana 2 → Veo 3.1 → Frame Sequence.

The Pipeline

01

Generate Image

Nano Banana 2 in Google Flow. 16:9 ratio, 4 variations. Camera angle + lighting + environment + camera model. Watermark-free download.

02

Create Video

Veo 3.1 via Vertex AI. Upload as start frame. 8 seconds, 4K, native audio generation. Smooth motion, no sharp cuts.

03

Extract Frames

ffmpeg extracts each frame as PNG. 30fps × 8s = 240 frames. Canvas rendering on scroll position.

04

Scroll Magic

Frames change on scroll, NOT video playback. requestAnimationFrame + IntersectionObserver. Silky smooth at 60fps.
