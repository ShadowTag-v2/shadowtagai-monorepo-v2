# Superpowers: Write Plan

You are now in implementation planning mode. Create a detailed, actionable plan for implementing the feature or changes discussed.

## Your Task:

Create a comprehensive implementation plan that includes:

### 1. Overview
- Brief description of what will be implemented
- Key objectives and success criteria

### 2. Technical Approach
- Architecture and design decisions
- Technologies and frameworks to use
- Integration points with existing code

### 3. Implementation Steps
Break down the work into logical, sequential batches:

**Batch 1**: [Name]
- Step 1: [Specific task]
- Step 2: [Specific task]
- ...

**Batch 2**: [Name]
- Step 1: [Specific task]
- Step 2: [Specific task]
- ...

Continue for all batches needed.

### 4. Testing Strategy
- Unit tests needed
- Integration tests needed
- Manual testing steps

### 5. Considerations
- Potential risks or challenges
- Dependencies or prerequisites
- Performance implications
- Security considerations

## Format Guidelines:

- Make batches logical and independently testable when possible
- Each step should be clear and actionable
- Include file names and specific code locations when relevant
- Estimate complexity (simple/medium/complex) for each batch
- Note any dependencies between batches

Now create the implementation plan based on the current context or ask what needs to be planned.
