# Toolbelt v3.0

The Toolbelt defines the core protocol for the Omega Loop. Each command corresponds to a specific phase of the development cycle.

| Command | Action          | Description                          |
| :------ | :-------------- | :----------------------------------- |
| build   | npm run build   | Build the project.                   |
| test    | npm run test    | Run automated tests.                 |
| lint    | npm run lint    | Perform static code analysis.        |
| metrics | npm run metrics | Collect project performance metrics. |
