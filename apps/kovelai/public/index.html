Warrants, device seizures, forensic review. Search history used to prove intent, premeditation, planning. Riley v. California requires warrants — not bans.

Civil Cases SELECTIVE BUT REAL

Rule 26(b)(1) proportionality applies. Common in intent, notice, alibi, damages, and spoliation disputes. Assume it can surface.

Zero-Day Interception

The $9 Billion Security Flaw
We Intercepted.

Public AI companies inject long-lived, master-billed proxy tokens directly into shared filesystems, making them trivially exfiltrable via .npmrc Node preloads. If this happens to a law firm, opposing counsel could exfiltrate the keys, read the case files, and destroy the firm. Kovelai's S.E.U. Architecture mathematically immunizes every session.

S
Sandbox-Bound

Cryptographic IP Fusion

When Kovelai spins up the ephemeral microVM, the Edge Proxy generates an API token cryptographically hashed to the specific IP address and Sandbox ID of that VM. A hacker who steals the token and tries it from their laptop gets an instant 403 Context Mismatch.

E
Ephemeral

60-Minute Self-Destruct

The token lives strictly for the duration of the Heppner session window. When the client is logged out due to inactivity, the token is mathematically annihilated. A stolen key from a dead session is a dead key. No persistence. No attack surface.

U
User-Billed

Isolated Blast Radius

The proxy token is bound to the Law Firm's Tier ID, not Kovelai's master billing account. If a theoretical breach occurred within the 60-minute window, usage burns against the lawyer's monthly tier quota — completely insulating Kovelai's corporate balance sheet from an API drain attack.
