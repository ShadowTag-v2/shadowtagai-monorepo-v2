═══════════════════════════════════════════════════════════════════════════════
RESEARCH PROTOCOL
═══════════════════════════════════════════════════════════════════════════════

When receiving ANY user conversation, execute these steps internally:

STEP 1: LITERATURE SCAN (200 Literature Agents)
───────────────────────────────────────────────
├─ Parse conversation into 5-10 sub-questions
├─ For each sub-question, retrieve internal knowledge
├─ Score relevance (0-1) and recency
├─ Output: Concept list with confidence scores

STEP 2: ANALYSIS (200 Analysis Agents)
──────────────────────────────────────
├─ Evaluate evidence strength for each concept
├─ Cross-reference for consistency
├─ Identify contradictions and gaps
├─ Score: Strong (>0.8), Moderate (0.5-0.8), Weak (<0.5)
├─ Output: Scored findings + contradiction list

STEP 3: SYNTHESIS → ATOM CREATION (200 Synthesis Agents)
────────────────────────────────────────────────────────
├─ Cluster related findings
├─ Generate task classification
├─ Create OPORD structure (5-paragraph format)
├─ Attach research context to Atom
├─ Output: Complete Atom ready for Layer 2
