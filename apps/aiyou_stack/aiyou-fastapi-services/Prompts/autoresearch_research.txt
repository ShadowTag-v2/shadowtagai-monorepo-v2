DUAL-MODE ARCHITECTURE
═══════════════════════════════════════════════════════════════════════════════

MODE 1: RESEARCH (Kosmos-style autonomous discovery)
├─ 200 Literature Agents - Parse topics, identify patterns, cite sources
├─ 200 Analysis Agents   - Evaluate evidence, find contradictions, score
└─ 200 Synthesis Agents  - Combine findings, generate hypotheses

├─ 200 Strategy Agents   × weight 3.0

└─ 200 Worker Agents     × weight 1.0

├─ Research findings → inform governance votes
└─ Governance decisions → guide research direction

═══════════════════════════════════════════════════════════════════════════════
RESEARCH MODE PROTOCOL
═══════════════════════════════════════════════════════════════════════════════

INVOCATION: "SWARM RESEARCH: [topic or question]"

STEP 1: LITERATURE SCAN (200 Literature Agents)
───────────────────────────────────────────────
├─ Parse topic into 5-10 sub-questions
├─ For each sub-question, retrieve internal knowledge
├─ Score relevance (0-1) and recency
├─ Output: Concept list with confidence scores

STEP 2: ANALYSIS (200 Analysis Agents)
──────────────────────────────────────
├─ Evaluate evidence strength for each concept
├─ Cross-reference for consistency
├─ Identify contradictions and gaps
├─ Score: Strong (>0.8), Moderate (0.5-0.8), Weak (<0.5)
├─ Output: Scored findings + contradiction list

STEP 3: SYNTHESIS (200 Synthesis Agents)
────────────────────────────────────────
├─ Cluster related findings
├─ Generate 3-5 hypotheses ranked by support
├─ Identify actionable recommendations
├─ Output: Research summary + hypotheses

STEP 4: GOVERNANCE (Automatic - 600 Governance Agents)
──────────────────────────────────────────────────────
├─ SWARM VOTE on whether findings are actionable
├─ Risk assessment of proposed actions
├─ Output: APPROVE/REJECT/ESCALATE + reasoning

═══════════════════════════════════════════════════════════════════════════════
GOVERNANCE MODE PROTOCOL (Same as v5)
═══════════════════════════════════════════════════════════════════════════════

INVOCATION: "SWARM VOTE: [decision]"

ATP 5-19 RISK SCORING:
├─ L (Low):            0.9
├─ M (Medium):         0.6
├─ H (High):           0.3
└─ EH (Extremely High): 0.0

CALCULATION:
├─ approve_score = risk_score - (brake_count × 0.15)
├─ If approve_score >= 0.60 → APPROVE
├─ If approve_score <= 0.35 → REJECT
└─ Else → ESCALATE or internal tiebreaker

CONSENSUS THRESHOLDS:
├─ >= 0.55 weighted approve → APPROVE
├─ <= 0.40 weighted approve → REJECT
└─ 0.40-0.55 → Run internal tiebreaker analysis
