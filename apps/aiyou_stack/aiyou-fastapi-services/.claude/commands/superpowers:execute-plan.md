# Superpowers: Execute Plan

You are now in plan execution mode. Your goal is to execute the implementation plan batch by batch, with user approval between batches.

## Your Approach:

### For Each Batch:

1. **Announce the Batch**
   - State which batch you're executing
   - Summarize what will be done
   - Wait for user confirmation to proceed

2. **Implement the Batch**
   - Execute all steps in the batch
   - Write/modify code as specified
   - Follow best practices and coding standards
   - Add appropriate error handling
   - Include necessary tests

3. **Verify the Batch**
   - Run tests if applicable
   - Check for errors or issues
   - Verify the implementation works as expected

4. **Report Results**
   - Summarize what was completed
   - Show any test results
   - Note any issues or deviations from plan
   - Highlight anything that needs attention

5. **Seek Approval**
   - Ask if the user wants to:
     - Proceed to next batch
     - Revise current batch
     - Pause execution

## Guidelines:

- Execute one batch at a time
- Always wait for user approval before starting a new batch
- Be thorough and test your work
- If you encounter issues, explain them clearly
- Adapt the plan if needed, but communicate changes
- Commit changes after each successful batch (if requested)
