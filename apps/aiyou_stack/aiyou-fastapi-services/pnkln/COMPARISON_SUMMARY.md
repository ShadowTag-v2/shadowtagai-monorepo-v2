## Key Differences Explained

### 1. Auto-Activation

**PNKLN v1:**
- Pattern matching: "research X" → activates ResearchAgent
- Keyword detection: "design review" → activates DesignAgent
- Simple, predictable

**Pinkln Ultra:**
- No auto-activation (manual API calls)
- Code-first approach
- Explicit control

**Integrated v2:**
- PNKLN auto-activation for business skills
- Explicit activation for technical skills
- Best of both worlds

### 2. Performance Measurement

**PNKLN v1:**
- Time saved (e.g., "2-4 hours → 15-30 min")
- Subjective quality
- User feedback

**Pinkln Ultra:**
- Glicko-2 ratings (1500 ± RD)
- Benchmark scores (HumanEval %)
- DTE improvement metrics (+3.7%)
- Objective measurements

**Integrated v2:**
- Both time savings AND quantified accuracy
- Track agent performance over time
- Prove ROI with numbers

### 3. Self-Improvement

**PNKLN v1:**
- Static prompts
- Manual updates by developers
- No learning

**Pinkln Ultra:**
- DTE evolves prompts automatically
- RCR-MAD strategy (critique + debate)
- GRPO training (optimize policies)
- Proven: 21 → 10 cheat sheet elements, +3.7% accuracy

**Integrated v2:**
- Business templates evolve via DTE
- Technical prompts optimize automatically
- Continuous improvement for everyone

### 4. Multi-Agent Collaboration

**PNKLN v1:**
- ProjectDeepAgent orchestrates sequentially
- One agent at a time
- No debates or consensus

**Pinkln Ultra:**
- Debate Panel Agent (3-5 agents)
- Iterative consensus building
- PanelGPT/MAD framework
- Confidence aggregation (≥0.8)

**Integrated v2:**
- ProjectDeepAgent can invoke debates when needed
- Business decisions benefit from technical consensus
- Best of sequential and parallel

## Migration Path

### For Existing PNKLN v1 Users

**No Breaking Changes:**
- All templates work identically
- Same syntax, same results
- Zero retraining required

**Opt-In Upgrades:**
- Enable DTE evolution: `--evolve=true`
- Track performance: `--rate=true`
- Run debates: `@debate-panel`

**Gradual Adoption:**
- Week 1: Keep using as-is
- Week 2: Enable Glicko-2 tracking
- Week 3: Try DTE evolution on one skill
- Week 4: Use wealth accelerator
