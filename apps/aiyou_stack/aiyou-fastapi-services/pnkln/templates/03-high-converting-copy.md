# Template 03: High-Converting Copy

## Purpose
Generate persuasive, conversion-optimized copy using proven psychological triggers.

## When to Use
- Creating landing pages
- Writing email sequences
- Crafting ad copy
- Optimizing CTAs
- Improving conversion rates

---

## Copy-Paste Prompt

```
@copy [COPY TYPE]

Create high-converting copy with these parameters:

**Copy Type**: [Landing Page / Email / Ad / CTA / Social Post / Sales Page]
**Product/Service**: [Detailed description of what you're selling]
**Target Audience**: [Specific persona with pain points]
**Primary Benefit**: [Core value proposition - be specific and measurable]

**Psychological Triggers to Use**:
- [ ] Scarcity (limited availability)
- [ ] Social Proof (others' success)
- [ ] Authority (credibility markers)
- [ ] Reciprocity (give value first)
- [ ] Curiosity (open loops)
- [ ] Loss Aversion (what they'll miss)

**Tone**: [Professional / Casual / Urgent / Empathetic / Authoritative]

**Constraints**:
- [Any word limits, brand guidelines, or restrictions]

**Deliverables**:
- Primary copy (headline, subheadline, body, CTA)
- 3 A/B test variants
- Trigger analysis (what triggers used and why)
- Conversion hypothesis (expected lift)

**Success Metrics**:
- [What conversion means for this copy]
```

---

## Example Usage

```
@copy Landing Page

Create high-converting copy with these parameters:

**Copy Type**: Landing Page
**Product/Service**: AI-powered workflow automation platform that reduces engineering time by 60% through autonomous agent orchestration
**Target Audience**: Technical founders and CTOs at Series A startups who are overwhelmed with technical debt and feature requests

**Primary Benefit**: Ship 3x more features without hiring more engineers

**Psychological Triggers to Use**:
- [X] Scarcity (limited beta slots)
- [X] Social Proof (other startups using it)
- [X] Authority (backed by Y Combinator)
- [X] Curiosity (how it works)
- [ ] Reciprocity
- [X] Loss Aversion (falling behind competitors)

**Tone**: Professional but approachable - technical enough for credibility, simple enough to skim

**Constraints**:
- Headline: max 15 words
- Subheadline: max 25 words
- Body: scannable with short paragraphs
- Must include specific metrics

**Deliverables**:
- Primary copy (headline, subheadline, body, CTA)
- 3 A/B test variants
- Trigger analysis (what triggers used and why)
- Conversion hypothesis (expected lift)

**Success Metrics**:
- Primary conversion: Book demo call
- Secondary conversion: Sign up for beta waitlist
```

## Tips for Best Results

1. **Be Specific About Audience**: "CTOs at Series A" not "technical people"
2. **Quantify Benefits**: "60% faster" not "much faster"
3. **Choose Right Triggers**: Match triggers to audience psychology
4. **Test Variants**: Always generate A/B test options
5. **Define Success**: Specify what conversion means
