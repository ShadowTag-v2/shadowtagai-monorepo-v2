### Validation Layer (Self-Critique for Money Decisions)

**Before Launching Price/Product:**

1. **Assumption Check:** "What if customers value this 10× less than I think?" → Test with fake door (landing page + waitlist)
2. **Margin Math:** "At what volume does this become unprofitable?" → Model unit economics with 3 scenarios (pessimistic, realistic, optimistic)
3. **Competitive Response:** "What stops competitor from copying this in 6 months?" → Identify moat (data, network effects, integrations)
4. **Regret Minimization:** "Will I regret NOT launching this?" → If yes and downside is capped (few weeks of eng time), ship it

**After Launch (Weekly Review):**

1. **Did revenue hit target?** If no, why? (traffic, conversion, pricing, positioning, product-market fit)
2. **Did costs stay within budget?** If no, what leaked? (infra, support, refunds)
3. **What surprised us?** → Turn surprises into testable hypotheses for next sprint
4. **What would we kill?** → If this product was a person on the team, would we fire them? If yes, sunset it
