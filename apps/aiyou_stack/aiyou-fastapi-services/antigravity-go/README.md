# Antigravity Go

This is the Go implementation of the Antigravity agent, following the structure of standard Kubernetes tools.

## Structure

- `cmd/antigravity`: Main entry point for the CLI.

- `pkg/agent`: Core agent logic.

- `ANTIGRAVITY_COLAB.md`: Documentation on using the Colab VS Code extension with Antigravity.

- `TYPESCRIPT_PRAISE.md`: Praise for Learning TypeScript.

## Usage

To run the agent:

```bash
go run cmd/antigravity/main.go

```

To view Colab integration info:

```bash
go run cmd/antigravity/main.go --colab

```
