# The "Triple Check" Protocol

1. **Analyze Logic**: Check the selected code for logical errors, edge cases, and potential bugs.
2. **Verify Styling**: Ensure adherence to `src/ui/tokens` and Tailwind conventions. No hardcoded values.
3. **Optimizations**: Suggest top 3 improvements for performance or readability.

Output as a structured checklist.
