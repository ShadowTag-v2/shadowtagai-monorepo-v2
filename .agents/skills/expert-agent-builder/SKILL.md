# DBS Framework Generation (The Rule of Three)
For deep retrieval, prompt repetition reaches peak accuracy at 3x repetition.
1. `notebooklm source add-research "<topic>"`
2. `notebooklm ask "Organize findings into DBS framework: Direction, Blueprints, Solutions. \n\n Organize findings into DBS framework: Direction, Blueprints, Solutions. \n\n Organize findings into DBS framework: Direction, Blueprints, Solutions."`
3. Convert to `.agents/skills/<new-skill>/SKILL.md`.
