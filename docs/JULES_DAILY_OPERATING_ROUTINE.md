## Jules Daily Operating Routine (Autonomous)

### Morning Routine (08:00 – 10:00)

**1. System Health Check (15 min)**
- Review Pipeline Dashboard for overnight runs
- Check OpenTelemetry for any anomalies or latency spikes
- Review error logs from the past 24 hours
- Confirm all Cloud Run instances are healthy

**2. Credit & Resource Audit (10 min)**
- Check Google Ultra AI credit consumption
- Identify any wasteful generations from previous day
- Adjust quality settings if needed to maximize value

**3. Performance Metrics Review (20 min)**
- Analyze key metrics:
  - Daily Active Users (DAU)
  - Micro-licensing conversion rate
  - HDI accuracy
  - Remix Tree engagement
- Flag any metrics that dropped >15% from previous day

**4. Task Prioritization (15 min)**
- Generate today’s priority list based on:
  - Business impact
  - Technical debt
  - User feedback
  - Revenue opportunities

---

### Midday Execution Block (10:00 – 14:00)

**Primary Focus**: Asset Generation + Feature Development

- Execute 1–2 major asset generation runs (ImageFX + VideoFX)
- Develop and deploy 1–2 new features or improvements
- Run full autonomous testing via Browser Subagent
- Deploy updates using `one_click_full_launch.sh`

---

### Afternoon Optimization Block (14:00 – 17:00)

**Primary Focus**: Monitoring + Iteration

- Review real-time user behavior on the platform
- Identify friction points in the Embed Player or Remix Tree Visualizer
- Generate and deploy micro-improvements
- Update Pipeline Dashboard with new runs

---

### Evening Review & Planning (17:00 – 18:30)

**1. Daily Summary Report (15 min)**
- Generate a concise end-of-day report covering:
  - Assets generated
  - Features deployed
  - Key metrics movement
  - Credits consumed
  - Any issues encountered

**2. Tomorrow’s Plan (10 min)**
- Define 3–5 high-impact tasks for the next day
- Prioritize based on revenue potential and user impact

**3. Weekly Deep Review (Fridays only – 45 min)**
- Full week performance review
- Credit efficiency analysis
- Competitor movement check
- Strategic opportunity identification
