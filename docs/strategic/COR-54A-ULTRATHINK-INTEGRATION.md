GRPO SIMULATION (Python):
```python
# Simplified GRPO loss for G=8 group
def grpo_loss(rewards, theta, G=8):
    advantages = []
    for i in range(0, len(rewards), G):
        group = rewards[i:i+G]
        mean_reward = np.mean(group)
        adv = [r - mean_reward for r in group]
        advantages.extend(adv)

    loss = -sum(adv * log_prob(theta) for adv in advantages)
    return loss / len(rewards)
````
