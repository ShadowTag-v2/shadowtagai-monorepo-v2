# Common Tasks

## Start a session

1. GET /api/hydrate-pack
2. Load authority-current.json
3. Load top atoms
4. Load active tasks and drift reports
5. Inspect changed files only
6. Upgrade code toward authority if mismatch exists

## Update standards/settings

1. Update authority-current.json
2. Persist authority snapshot
3. Rebuild authority atoms
4. Append journal entry
5. Regenerate memory bank derived files
6. Run drift detector
