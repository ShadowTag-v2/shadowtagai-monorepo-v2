# Cinematic Verification Loop

## Purpose

Convert end-to-end software work into evidence-bearing artifacts rather than ungrounded success claims.

## Loop

1. Build.
2. Run.
3. Record.
4. Upload artifact.
5. Judge with multimodal model.
6. Pass or fail.
7. Auto-heal or escalate.
8. Preserve proof artifact with the change.

## Artifact types

- UI-run video.
- Screenshots.
- Logs.
- Test output.
- Structured pass/fail summary.

## Default judgment standard

- No "looks good" assertions without artifact evidence.
- No mergeable output without a verifiable proof record.
