### Edge Inference (Per Request)

**Assumptions:**

- Average request: 500 tokens in + 200 tokens out

- Model: Llama-3-70B-Instruct

- GPU: NVIDIA L40S ($2.80/hr reserved)

```

GPU time per inference: 12ms
Requests/hour at 100% util: 300,000
GPU cost per request: $2.80 / 300,000 = $0.0000093

Other costs:

- Bandwidth: $0.0000050

- Storage: $0.0000010

- Orchestration: $0.0000020
Total cost: $0.0000173

Revenue per request: $0.00144
Gross profit: $0.00142
Gross margin: 98.8%

```

**At scale (20B requests/month in 2030):**
