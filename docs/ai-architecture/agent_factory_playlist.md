# The Agent Factory Playlist Summaries

### **Playlist: The Agent Factory**

**Focus:** Building production-ready AI agents, tools, and architectures.

#### **1. Agent sandbox and Pod snapshots: Supercharging agents on GKE**


* **Topic:** Running AI agents securely and efficiently on Google Kubernetes Engine (GKE).

* **Key Technologies:** GKE, Agent Sandbox, Pod Snapshots.

* **Summary:**

    * **Where to Run Agents:** GKE recommended for high-scale or custom usage.

    * **Agent Sandbox:** Strong isolation for untrusted code execution.

    * **Pod Snapshots:** Reduces startup latency from seconds to milliseconds.

#### **2. Building with Gemini 3, AI Studio, and Nano Banana**


* **Topic:** New Google AI tools.
