# Component Specification: TACSOP 0 Frontend Clone

## Overview
This specification details the implementation plan for the TACSOP 0 frontend clone module.

## Requirements
- Use Next.js 16, Tailwind v4, and shadcn/ui.
- Follow the 5-phase cloning pipeline (Recon → Foundation → Specs → Parallel Build → Assembly).

## Components
1. **AppLayout**: Main layout wrapper.
2. **HeroSection**: Primary hero section.
3. **Dashboard**: Main dashboard view.
