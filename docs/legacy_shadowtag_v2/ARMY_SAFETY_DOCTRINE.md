# JUDGE 6: UNIFIED SAFETY DOCTRINE (V2.0.0)
**Project:** shadowtag-omega-v2
**Scope:** Total Governance

## THE SIX-GATE PIPELINE
1. **Ingest:** Map inputs to METT-TC.
2. **Initial Scoring:** Risk = Probability x Severity.
3. **Control Injection:** Apply Admin controls.
4. **Residual Scoring:** Re-calculate Risk.
5. **Authority Check:** GREEN (Auto), YELLOW (Manager), RED (STOP).
6. **Commit:** Execute.
