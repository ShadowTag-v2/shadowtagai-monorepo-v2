# JUDGE 6: CYBERSECURITY RISK MANAGEMENT FRAMEWORK (RMF)
**Source:** DoDI 8510.01 (Effective July 19, 2022) & NIST SP 800-37

## I. THE 7-STEP LIFECYCLE
1.  **PREPARE:** Organization-level and System-level preparation. Assign roles (AO, ISSM).
2.  **CATEGORIZE:** Categorize system per CNSSI 1253 (Confidentiality, Integrity, Availability).
3.  **SELECT:** Select NIST SP 800-53 controls based on categorization.
4.  **IMPLEMENT:** Deploy controls and document in the Security Plan.
5.  **ASSESS:** Verify controls are effective (Security Assessment Report).
6.  **AUTHORIZE:** The Authorizing Official (AO) grants Authorization to Operate (ATO).
7.  **MONITOR:** Continuous monitoring of security posture (ConMon).

## II. KEY ROLES
* **AO (Authorizing Official):** Accepts risk. (Judge #6 acts as Digital AO).
* **ISSM:** Responsible for cybersecurity program.
* **SCA:** Independent validator.

## III. DECISIONS
* **ATO:** Go live.
* **ATO-C:** Conditional (POA&M required).
* **DATO:** Denial.
