# Global Defaults (Antigravity)
1. **Model**: Always default to `gemini-3.1-family` logic unless specified otherwise.
2. **Language**: Always use TypeScript for frontend, Python (3.12+) for backend.
3. **Endpoint**: Use the official Google Cloud Vertex AI endpoints.
4. **Accuracy**: After adding a feature, self-evaluate accuracy against the request and give a score out of 100.
