# Superpowers: Execute Plan

**Trigger:** `/superpowers:execute-plan`

## Objective
To autonomously execute the approved plan using **Antigravity (Ballistic Engine)** with zero hesitation.

1.  **Read Plan:** Load `implementation_plan.md`.
2.  **Generate Code:** Write the actual implementation files (`src/api/health.py`, etc.).
3.  **Judge Verification:** Pass every file write through the Judge's "Brakes" check (Secrets? Safety?).
4.  **Verify:** Run the verification steps (e.g., `curl localhost/healthz`).
5.  **Log Success:** Update `task.md` and `.beads/issues.jsonl`.
6.  **Celebrate:** "Another dent in the universe."

**Outcome:** Functional, production-ready code.
