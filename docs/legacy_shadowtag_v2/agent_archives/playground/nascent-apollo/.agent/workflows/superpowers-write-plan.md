# Superpowers: Write Plan

**Trigger:** `/superpowers:write-plan`

## Objective
To crystallize the brainstorming session into a concrete, executable `implementation_plan.md` using **Beads (Institutional Memory)**.

## Steps
1.  **Recall Context:** Retrieve the design summary from the brainstorm phase.
2.  **Draft Plan:** Create/Update `implementation_plan.md` with:
    *   **Goal:** The specific outcome.
    *   **Proposed Changes:** Exact files and logic.
    *   **Verification:** How we will prove it works (The Maiden Voyage).
3.  **Governance Check:** Ensure the plan respects "Purpose, Reasons, Brakes" (JudgeSix).
4.  **Store Memory:** Log this decision in `.beads/issues.jsonl`.

**Outcome:** A signed-off plan ready for execution.
