# Superpowers: Brainstorm

**Trigger:** `/superpowers:brainstorm "Topic"`

## Objective
To initiate a deep-dive design session using the **Titans (Neural Context)** to explore architecture, constraints, and "Ultrathink" possibilities before any planning begins.

## Steps
1.  **Analyze Request:** Take the user's input topic.
2.  **Activate Titans:** Use the "Steve Jobs" persona to question assumptions ("Why must it function so?").
3.  **Propose Options:** Generate 3 distinct approaches (Safe, Bold, Ultrathink).
4.  **Debate:** Use internal PanelGPT (Security vs. Architect) to vet ideas.
5.  **Output:** A concise design summary ready for the "Write-Plan" phase.

**Note:** Do not write code yet. Just design.
