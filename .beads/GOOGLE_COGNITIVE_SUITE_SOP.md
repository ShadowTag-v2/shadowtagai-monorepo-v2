This document serves as the absolute Source of Truth for generating assets outside the IDE.

## 1. Google Stitch (Ideate & Redesign)
- **Vibe Designing:** Use Stitch to generate UI. Export `DESIGN.md` for consistent tokens (colors, typography).
- **Ideate Agent:** Use at the start of a project. Describe a problem (e.g., "Analyze top 3 e-commerce flows"). Set creative range slider to "YOLO" for divergent exploration.
- **Redesign Agent (Nano Banana Pro):** Use to visually iterate on UI/UX designs before converting to code. Upload an image/sketch, describe changes, review artifact, then "Convert to Code". Export to Figma if needed.

## 2. Google Opal (Watermark-Free Veo 3.1)
- **Workflow:** Navigate to Opal. Create New -> Add "Generate" node -> Select "VEO 3.1" agent. Set aspect ratio (16:9 or 9:16). Type prompt. Preview -> Start.
- **Image-to-Video:** Add "User Input" node, connect to "Generate" node. Set input type to "Image". Upload reference image (generated by Nano Banana 2). Prompt: "Animate this perfectly."
- **Result:** MP4 downloads with NO watermark. Crucial for cinematic hero backgrounds.

## 3. Google Vids (Veo 3.1 Workspace)
- **Workflow:** Select "Create from scratch" or "Animate an image". Set aspect ratio. Uses Veo 3.1 to generate clips directly into a timeline.

## 4. Google Flow & Whisk
- Use Flow for cinematic AI filmmaking. Frame-by-frame context preservation.
- Use Whisk for rapid asset and mood board generation.

## 5. Antigravity & Stitch MCP Integration
- Use the installed `stitch-skills` (`stitch-design`, `react:components`, `stitch-loop`).
- Antigravity uses Stitch MCP to read the generated UI and write the frontend code natively, eliminating manual copy-pasting.
