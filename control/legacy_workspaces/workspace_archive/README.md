# ChatGPT Archive

Repository for ChatGPT conversation exports and code extracted from conversations.

## Structure

```
chatgpt-archive/
├── conversations/     # Raw conversation exports
├── extracted_code/    # Code blocks extracted from conversations
└── projects/          # Organized project code from ChatGPT
```

## Usage

Point ChatGPT at this repo to contribute code:
- Repo: https://github.com/ehanc69/chatgpt-archive
- Use GitHub API or paste code for Claude to commit
