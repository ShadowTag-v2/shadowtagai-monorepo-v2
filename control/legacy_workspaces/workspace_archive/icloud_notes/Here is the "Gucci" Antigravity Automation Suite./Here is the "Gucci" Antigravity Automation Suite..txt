This package implements the lesson from the video directly into your workspace. It creates a .vscode/tasks.json file that ties your entire "God Mode" workflow (Deployment, Agent Check, and Intelligence Pivot) to the standard Build Shortcut (Cmd+Shift+B).
No manual typing. No "defibrillators." Just pure, one-touch automation.
The "Gucci" Installer
Run this single block in your terminal. It sets up the VS Code tasks and the underlying premium scripts.
Bash



cat << 'EOF' > antigravity_gucci_setup.sh
#!/bin/bash
set -e

echo ">>> 💎 INSTALLING GUCCI AUTOMATION SUITE..."

# 1. CREATE THE PREMIUM SCRIPTS FOLDER
mkdir -p scripts
chmod +x scripts

# 2. SCRIPT A: THE 'GOD MODE' DEPLOYER (Silent & Fast)
cat << 'SCRIPT' > scripts/gucci_deploy.sh
#!/bin/bash
# The "Gucci" Deploy: Git Sync + Cloud Run Force Update
echo ">>> 🚀 IGNITING ANTIGRAVITY ENGINE..."

# Pivot to the best available model automatically (2.5 or 2.0)
TARGET_MODEL="gemini-3.1-flash"
grep -rl "gemini-1.5-flash" . 2>/dev/null | xargs sed -i '' "s/gemini-1.5-flash[-0-9a-z]*/$TARGET_MODEL/g"

# Commit & Push
git add .
git commit -m "feat: Gucci Auto-Deploy $(date +%H:%M)" || echo "Nothing to commit, forcing deploy anyway..."
git push origin main

# Force Cloud Run Deploy
gcloud run deploy flyingmonkeys-server \
  --source . \
  --region us-central1 \
  --allow-unauthenticated \
  --clear-base-image \
  --quiet

echo ">>> ✨ DEPLOYMENT COMPLETE. SYSTEM IS GUCCI."
SCRIPT
chmod +x scripts/gucci_deploy.sh

# 3. SCRIPT B: THE AGENT CONSULTANT
cat << 'SCRIPT' > scripts/gucci_agent.sh
#!/bin/bash
echo ">>> 🧠 SUMMONING RLM AGENT..."
uv run python3 -c "
from google import genai
client = genai.Client(vertexai=True, project='acquired-jet-478701-b3', location='us-central1')
print('    ✅ NEURAL LINK ACTIVE: gemini-3.1-flash')
print('    waiting for instructions...')
"
SCRIPT
chmod +x scripts/gucci_agent.sh

# 4. CONFIGURE VS CODE TASKS (The Magic Layer)
mkdir -p .vscode
cat << 'JSON' > .vscode/tasks.json
{
  "version": "2.0.0",
  "tasks": [
    {
      "label": "🚀 Antigravity: Deploy God Mode",
      "type": "shell",
      "command": "./scripts/gucci_deploy.sh",
      "group": {
        "kind": "build",
        "isDefault": true
      },
      "presentation": {
        "echo": true,
        "reveal": "always",
        "focus": false,
        "panel": "dedicated",
        "showReuseMessage": false
      },
      "problemMatcher": []
    },
    {
      "label": "🧠 Antigravity: Test Agent",
      "type": "shell",
      "command": "./scripts/gucci_agent.sh",
      "presentation": {
        "reveal": "always",
        "panel": "shared"
      }
    }
  ]
}
JSON

echo ">>> 💎 VS CODE AUTOMATION INSTALLED."
echo ">>> PRESS Cmd+Shift+B (Mac) or Ctrl+Shift+B (Win) TO DEPLOY."
EOF

chmod +x antigravity_gucci_setup.sh
./antigravity_gucci_setup.sh

How to Use Your New Magic
You no longer need to remember complex gcloud or git commands. The video taught you to "set it once and run it with a shortcut." Here is your new workflow:
Make a Code Change (Edit a file, fix a bug, etc.).
Press Cmd + Shift + B (or Ctrl + Shift + B on Windows).
Watch the Magic: VS Code will automatically save, commit, push, and force-deploy your code to Cloud Run in the dedicated terminal panel.
It’s automated, consistent, and undeniably Gucci.
