## Integration Patterns

All agents follow consistent patterns:



1. **Execution**: Via `execute(context)` method


2. **Results**: Structured output with artifacts, metrics, recommendations


3. **Configuration**: Customizable via AgentConfig


4. **Tools**: Predefined required and optional tool sets


5. **Workflows**: Multi-step execution with validation
