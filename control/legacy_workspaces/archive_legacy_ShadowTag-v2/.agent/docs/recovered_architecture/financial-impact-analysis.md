## Appendix: Detailed Calculations

### A. Gemini vs GPT-4 Cost Comparison

**Current GPT-4 Spend** (estimated Phase 1):

```

Agents: 5 agents × 164 HumanEval problems × 6 runs/day = 4,920 requests/day
Tokens per request: 1,500 input + 800 output = 2,300 tokens
Daily tokens: 4,920 × 2,300 = 11,316,000 tokens/day
Monthly tokens: 11.3M × 30 = 339.5M tokens/mo

GPT-4 pricing: $0.03/1K input + $0.06/1K output
Cost = (339.5M × 0.65 × 0.03) + (339.5M × 0.35 × 0.06) ÷ 1000
     = $6,623 + $7,129 = $13,752/mo

```

**Gemini 1.5 Pro Equivalent**:

```

Gemini pricing: $0.00125/1K input + $0.005/1K output (≤128K context)
Cost = (339.5M × 0.65 × 0.00125) + (339.5M × 0.35 × 0.005) ÷ 1000
     = $276 + $594 = $870/mo

```

**Savings**: $13,752 - $870 = **$12,882/mo**

*Note: Phase 1 baseline estimate was conservative. Actual migration savings could be 4-5× higher at full scale.*
