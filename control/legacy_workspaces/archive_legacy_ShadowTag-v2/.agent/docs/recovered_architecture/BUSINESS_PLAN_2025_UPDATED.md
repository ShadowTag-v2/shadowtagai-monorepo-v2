### LLM Stack (4-LLM Orchestration)

**Allocation** (from Memory Thread Rollup):

```python
ALLOCATION = {
    'gemini': 0.40,      # Gemini 2.0 Flash (multimodal bulk processing)
    'claude': 0.35,      # Sonnet 4.5 (coordination)
    'gpt5': 0.15,        # Structured output, coding
    'perplexity': 0.05,  # Research, web-grounded
    'grok': 0.05         # Intake/decomposition only
}

ROUTING: Grok → Sonnet 4.5 → 3-LLM rotation (Gemini/GPT-5/Perplexity) → Claude Code synthesis

```

**Review Rotation**:

- Round 1: Each LLM answers assigned threads

- Round 2: Rotate right → peer review by different LLM

- Round 3: Rotate right again → second review

- Result: Every answer touched by all 3 LLMs twice (quality assurance)

**Cost per Query**: $0.08-0.12 (with review rounds)
