## 2. Cofounder Decision Matrix

### Core Strategic Frameworks



- **VRIO**: Stress-test resource value/rarity under disruption.


- **Value Stick**: Test WTP/WTS/Cost fluctuations via Monte Carlo.


- **Blue Ocean**: Eliminate/Reduce/Raise/Create analysis.


- **McKinsey Horizons**: H1 (Core), H2 (Emerging), H3 (Breakthrough).


- **Strategy Diamond**: Arenas, Vehicles, Differentiators, Staging, Economic Logic.

### Execution Philosophies



- **Tiny Teams Thesis**: Billion-dollar leverage with solo/lean execution.


- **Disciplined Entrepreneurship**: 24-step MIT structure.


- **Mochary’s CEO System**: Documented decisions with kill-switches.


- **Elon Musk Principles**: First principles, 80-100hr sprints, 10x advantage.

### Governance & Filters



1. **Business Judgment Rule**: Anchor for all decisions.


2. **ROI Thresholds**: ≥3x in 18 months.


3. **Kill-Switch Triggers**: Re-earn capital based on CAC/LTV/NPS.


4. **LTV:CAC**: ≥4:1 within 12-18 months.
