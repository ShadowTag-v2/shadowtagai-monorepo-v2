# Workflow: Superpowers Write Plan

**Trigger:** `/superpowers:write-plan`

**Goal:** Commit the strategy to Institutional Memory (Beads).

**Steps:**

1.  **Synthesize:** Take the brainstorm output.
2.  **Structure:** Create a Markdown plan (Step-by-Step).
3.  **Persist:** Write the plan to `.beads/issues.jsonl` using `BeadsEngine.log()`.
4.  **Verify:** Read it back to ensure it is saved.
