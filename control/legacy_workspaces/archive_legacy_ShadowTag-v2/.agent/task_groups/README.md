# Task Groups

Task Groups are high-level units of work used by the Antigravity Agent in **Planning Mode**.

## Usage

- **Runtime:** The agent automatically creates task groups when breaking down complex user prompts.
- **Structure:** Each task group contains subtasks, artifacts (plans), and verification steps.
- **Navigation:** Users can expand/collapse task groups in the Activity Bar to track progress.

## Pre-Defined Groups (Templates)

_Currently, Task Groups are primarily dynamic. Place any pre-defined plan templates in `.agent/workflows/` instead._
