# Decision: Monorepo OS Architecture
- **Date:** 2026-04-27
- **Status:** Accepted
- **Context:** Thread synthesis consolidated scattered pieces into one operating system.
- **Decision:** Adopt the 12-truth-surface Monorepo OS architecture with Beads for work,
  Memory for knowledge, GitNexus for code graph, Bazel for build, ToolGateway for safety,
  Dagger for execution, and AG-UI for the command deck.
- **Consequences:** All agents operate under this single doctrine. No more ad-hoc tooling.
